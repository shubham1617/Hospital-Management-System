## 🔗 Quick Links

### API Access
- **Base URL:** http://localhost:8081/doctor-service
- **Swagger UI:** http://localhost:8081/doctor-service/swagger-ui.html
- **API Docs:** http://localhost:8081/doctor-service/v3/api-docs

### Common Endpoints
- Create Doctor: `POST /api/v1/doctors`
- Get All: `GET /api/v1/doctors`
- Get by ID: `GET /api/v1/doctors/{id}`
- Update: `PUT /api/v1/doctors/{id}`
- Delete: `DELETE /api/v1/doctors/{id}`

See QUICK_REFERENCE.md for more endpoints and examples.
